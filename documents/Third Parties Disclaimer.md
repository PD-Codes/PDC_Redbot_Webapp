## Introduction
This Dashboard is a website that provides a friendly interface to control and modify how a Red-DiscordBot instance behaves in a Discord (discord.com) server. This Dashboard allows third-party integrations (for example, widgets and panels contributed by individual cogs) for more control and easier use. This document describes what may happen with such Third Party Integrations.

- "Owner," "we" or "us" refers to the person(s) who operate this self-hosted deployment.
- "You" refers to the person reading this disclaimer or accessing this website.

This document describes:

- How Third Party integrations are structured.
- What to do when malicious use is detected.

If you do not wish to interact with Third Party integrations, then do not use any components of this Dashboard that are marked as a Third Party Integration, or disable them.

## 1. How Third Party integrations are structured
When you use a widget, panel or route that is contributed by a cog other than the supported Dashboard companion cog, you are interacting with code that is provided by that cog. Whatever data you input there, or whatever actions you take there, are handled by that respective cog.

Because this code lives in a separate cog, it is outside the control of this Dashboard's core code. While such a cog may have been trustworthy at the time it was installed, it is possible for it to be changed afterwards. If you use any part of these integrations, you acknowledge that you have read this document and accept that the Dashboard core is not responsible for the behaviour of third-party cog code.

## 2. What to do when malicious use is detected
If you believe a Third Party integration is being used maliciously, please contact the owner of this deployment through the contact method they provide, describing what you observed and which integration is involved. Action will then be taken as appropriate.

## 3. Disputes
If you have any complaints, concerns or issues with this disclaimer, you should contact the owner of this deployment first.
